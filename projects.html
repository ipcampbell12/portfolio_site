Flask Projects

Board Game Flask

This is a flask application that stores household boardgame statistics. It features a database using sqlite and flask_sqlalchemy, mutliple templates that output boardgame data on the screen using the jinja templating language, and a number of SQL queries that display relevant aggregate data about game winners.
